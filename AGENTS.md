# Next.js: ALWAYS read docs before coding

Before any Next.js work, find and read the relevant doc in `node_modules/next/dist/docs/`. Your training data is outdated — the docs are the source of truth.

# Skills

This project has agent skills installed in `.agents/skills/`. Before working on a relevant task, load the appropriate skill:

- **`vercel-react-best-practices`** — React and Next.js performance optimization rules
- **`vercel-composition-patterns`** — React composition patterns
- **`web-design-guidelines`** — UI, accessibility, and UX rules
- **`streamdown`** — streaming markdown renderer (used in this project)
- **`deploy-to-vercel`** — deploying to Vercel
